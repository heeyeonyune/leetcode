## leetcode solutions (Javascript)
JavaScript solutions for LeetCode problems, focused on practicing core data structures/algorithms and getting more comfortable with JavaScript fundamentals.

## What’s in this repo
Each problem solution lives in its own .js file, usually named with the LeetCode problem number and title, for example:
- 1_twoSum.js
- 9_palindrome.js
- 13_roman_to_int.js

## Requirements
Node.js (any modern version should work)

## Goals
- Build consistent practice solving LeetCode problems
- Learn and reinforce common algorithm patterns (two pointers, stacks, sorting, prefix checks, etc.)
- Familiarize myself with JavaScript (syntax, array/string methods, objects, etc..)

## Progress
This repo will grow over time as more problems are added
